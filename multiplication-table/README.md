# 九九表を表示する

## ヒント

二重のfor文を書いてみよう

1の段, 2の段, 3の段…と、作ってから表示してみよう 

## 実行コマンド

JavaScript
```shell:
node main.js
```
Ruby
```shell:
ruby main.rb
```
Python
```shell:
python3 main.py
```
PHP
```shell:
php main.php
```
