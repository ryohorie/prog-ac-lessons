# コマンドライン引数で受け取ったjsonファイルを読み込み、名前：年齢を表示する

## ヒント

ファイルの読み込み方を調べよう

jsonファイルのパース(解析)の仕方を調べよう

パースしたデータの参照の仕方を学ぼう

## 実行コマンド

JavaScript
```shell:
node main.js ./profiles.json
```
Ruby
```shell:
ruby main.rb ./profiles.json
```
Python
```shell:
python3 main.py ./profiles.json
```
PHP
```shell:
php main.php ./profiles.json
```

## 実行結果

```shell:
佐藤:22
鈴木:34
高橋:42
田中:15
```
