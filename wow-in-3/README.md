# 1から50までの数字を表示する。ただし3のつく数字と3の倍数のときだけ"WOW"と表示する

## ヒント

3の倍数→3で割り切れる

3の付く数字→3という文字が含まれている

※変数の"型"を意識してみよう

## 実行コマンド

JavaScript
```shell:
node main.js
```
Ruby
```shell:
ruby main.rb
```
Python
```shell:
python3 main.py
```
PHP
```shell:
php main.php
```
